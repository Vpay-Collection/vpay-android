## 简介

Vpay安卓版是与[Vpay服务端](https://github.com/Vpay-Collection/VPay)配套使用的监控客户端。


## 特性

- 监控微信支付宝收款到账信息，并推送给服务端
- 长期后台驻留，自动重启服务
- 自动心跳，适合长期挂机使用
- 无需hook，更安全
- ......
